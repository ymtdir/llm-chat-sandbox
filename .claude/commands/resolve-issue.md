# Issueの解決

このコマンドは、GitHub Issueの内容を読み取り、ブランチ作成から解決まで自動で行います。

## 手順

### ステップ1: Issue番号の解析

1. 引数からIssue番号を取得
2. GitHubリポジトリの設定を確認

### ステップ2: issue-resolutionスキルの実行

1. **issue-resolutionスキル**をロード
2. スキルにIssue番号を渡して解決プロセスを実行

### ステップ3: 結果の報告

```
✅ Issue #[番号]を解決しました！

ブランチ: issue-[番号]-[説明]
コミット数: [数]
テスト結果: ✅

PR作成の準備ができました。
必要に応じて /create-pr を実行してください。
```
