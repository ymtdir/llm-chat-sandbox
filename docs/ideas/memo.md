# アプリアイデアメモ

## 概要

LINE や Slack での会話を通じて、AIが毎日自動で日記をつけてくれるサービスのプロトタイプ。

ユーザーは「日記を書く」という意識なく、AIと日常会話するだけで日記が生成される。

## ターゲットユーザー

- 毎日日記をつけたいが、何を書けばいいかわからない人
- 日記が続かない人
- 話し相手が欲しい人

## 主な特徴

### チャットベースの日記生成

- LINE / Slack をインターフェースとして利用
- ユーザーは普段通り会話するだけ
- AIが会話内容から日記を自動生成

### AIキャラクター設定

- 複数のキャラクターをサービス側で設定可能
- 設定項目: 性別、関係値、性格、名前など
- 大枠はクラス設計でポリモーフィズムを使い、細かな特徴はパラメータ注入で柔軟に対応
- コードでガチガチに固めず、拡張しやすい設計を重視

## 開発方針

### フェーズ

1. **バックエンド（ヘッドレス）** ← まずここから
2. **チャットUI** — DESIGN.md ベースでAIにUI生成させるアプローチを試す
3. **外部連携** — MVP完成後に LINE / Slack / Discord 連携を試作

### 補足

- UIは自前で作る（外部連携はMVP後）
- バックエンドをヘッドレスで先行実装し、フロントは後から接続

## スコープ

- これは正式なアプリではなく**プロトタイプ**
- 詳細な設計は `/setup-project` で別途ドキュメント化予定
